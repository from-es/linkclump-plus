CSS Layout Functionality Test

About

This page is a test page to verify the Linkclump Plus's link selection function within elements to which the following basic CSS layout properties are applied:

  • margin
  • padding
  • max-width
  • min-width
  • position

It confirms whether Linkclump Plus can correctly select links under various CSS layout settings.

Default Style


html, body {
	margin : 0;
	padding: 0;

	font-size: 16px;
}

html {
	background-color: #000;
}

body {
	background-color: #fff;
}

main {
	padding: 1rem 0;
}
